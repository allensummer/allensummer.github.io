opencv基本绘图

发表于 2016-05-04   |   分类于 opencv   |  

基本绘图功能有：

  • 用 Point 在图像中定义 2D 点
  • 用OpenCV的函数 line 绘 直线
  • 用OpenCV的函数 ellipse 绘 椭圆
  • 用OpenCV的函数 rectangle 绘 矩形
  • 用OpenCV的函数 circle 绘 圆
  • 用OpenCV的函数 fillPoly 绘 填充的多边形
